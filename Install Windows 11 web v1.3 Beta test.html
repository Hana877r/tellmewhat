Windows 11 Installation Guide

 

ขั้นตอนการติดตั้ง Windown 11 ขั้นพื้นฐาน

A step-by-step guide to install Windows 11

ภาษา: ไทย
ความต้องการระบบ อุปกรณ์ที่จำเป็น  ขั้นตอนการติดตั้งไฟล์  ขั้นตอนการติดตั้งผ่าน Flash Drive

ความต้องการของระบบ

  • โปรเซสเซอร์ 1 GHz หรือมากกว่า
  • RAM อย่างน้อย 4 GB
  • อุปกรณ์เก็บข้อมูล 64 GB ขึ้นไป
  • การ์ดจอที่รองรับ DirectX 12
  • TPM เวอร์ชัน 2.0
  • รองรับ Secure Boot
  • 1 GHz or faster processor with at least 2 cores
  • 4 GB RAM or more
  • 64 GB or larger storage device
  • Compatible graphics card with DirectX 12 or later
  • TPM version 2.0
  • Secure Boot capability

อุปกรณ์ที่จำเป็นต่อการติดตั้ง 

  1. Flash Drive ( อย่างน้อย 8GB )
  2. Download the Windows 11 Installation Media Tool from the official Microsoft website.
  3.  อินเทอร์เน็ต เพราะการติดตั้ง Windows 11 ในหน้าลงชื่อเข้าใช้จำเป็นต้องใช้อินเทอร์เน็ต
  4. Product Key Windows 11 หรือ หากไม่มีสามารถใช้แบบทดลองได้ฟรี แต่จะจำกัดบางฟังก์ชั่น 
  5. บัญชี Microsoft หากยังไม่มีบัญชี คลิกลงทะเบียนที่นี่ได้เลย 

    6. ลงทะเบียน 
        Download Windows 11

ขั้นตอนการติดตั้งไฟล์ลง Flash Drive

  1. ดาวน์โหลดไฟล์การติดตั้ง Windows 11

  2. Download the Windows 11 Installation Media Tool from the official Microsoft website.

  3. เปิดไฟล์ Windows 11 เลือกภาษาที่ต้องการและเวอร์ชันของระบบปฎิบัติการจากนั้นกด Next


  4. เลือกเมนู USB flash drive จากนั้นกด Next โปรแกรมจะทำการติดตั้งไฟล์ใช้เวลา 10-20 นาที ขึ้นอยู่กับความเร็วอินเทอร์เน็ต

ขั้นตอนการติดตั้ง Windows 11 ผ่าน Flash Drive   

  1. เสียบ Flash Drive ที่ช่อง USB เข้าคอมพิวเตอร์หรือโน้ตบุ๊ต  
  2. Download the Windows 11 Installation Media Tool from the official Microsoft website.


  3. รีสตาร์ท คอมพิวเตอร์และเข้าเมนู BIOS โดยบางอุปกรณ์เข้าไม่เหมือนกัน ( กด F2, F10, Delete หรือ ESC )   จากนั้นเลือกเมนู Boot Optionไปที่เมนู UEFI Boot Order คลิ๊กที่เมนูแรก และเลือก Flash Drive ที่ติดตั้งไว้


  4. บูทผ่านแฟลชไดรฟ์สำเร็จจากนั้นติดตั้ ในหน้าแรกแนะนำให้ติดตั้งเป็นภาษาอังกฤษ เลือก Time and currency format เป็น Thai  แล้วกด Next จากนั้นก็กด Install


  5. เลือกที่เมนู Custom: Install Winsows only (advanced)     


  6. เมื่อถึงเมนูนี้ให้เลือกไฟล์ที่ต้องการติดตั้ง Windows 11 (หากมีหลายไดร์ฟให้เลือกไดร์ฟที่มีความจุเยอะที่สุด)      แนะนำไดร์ฟต้องมีขนาด 100GB ขึ้นไป    จากนั้นกด Next อุปกรณ์จะติดตั้งใช้เวลาค่อนข้างนาน   


  7. เมื่อติดตั้งสำเร็จขั้นตอนต่อไปคือการ Set up ต่างๆของอุกรณ์ 
    • เลือกที่อยู่เป็นประเทศไทย ( Thailand )


    • เลือกแป้นพิมพ์แรกเป็น US 


    • และแป้นพิมพ์ที่  2 เป็น TH Thai Kedmanee จากนั้นเลือก Add Layout
  8. ขั้นตอนนี้เป็นขั้นตอนลงชื่อเข้าใช้จำเป็นต้องใช้อินเทอร์เน็ตและบัญชีที่ลงทะเบียนไว้แล้วเท่านั้น                                    ( windows 11 บังคับให้ลงชื่อเข้าใช้เท่านั้นไม่สามารถข้ามขั้นตอนนี้ได้ )  
    • ตั้งชื่ออุปกรณ์


    • ลงชื่อเข้าใช้ Microsoft ที่ลงทะเบียนไว้


    • เมื่อลงทะเบียนสำเร็จ เลือกเมนู Don't back up my files จากนั้นกด Next 


  9. ติดตั้ง Windows 11 สมบูรณ์
